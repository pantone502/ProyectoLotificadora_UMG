## 💡 Descripción del Proyecto
Este sistema permite la administración de proyectos inmobiliarios y sus lotes correspondientes. 
Fue desarrollado en **C++** utilizando una arquitectura orientada a objetos y el framework **EloquentORM** para la persistencia de datos en una base de datos **MySQL**.

## 🚀 Funcionalidades (CRUD)
* **Proyectos:** Registro, listado, edición y eliminación de proyectos.
* **Lotes:** Gestión de lotes asociados a proyectos específicos, control de precios y estados (Disponible, Vendido, Reservado).

## 🛠️ Tecnologías utilizadas
* **Lenguaje:** C++
* **Base de Datos:** MySQL
* **Librerías:** MySQL Connector/C++ y EloquentORM

## 📋 Requisitos para ejecución
1. Servidor MySQL activo (XAMPP/Workbench).
2. Base de datos `lotificadora_db` creada (ver `script.sql`).
3. Librería `libmysql.dll` en la carpeta del ejecutable.
